# 🎬 AI Video Scene Builder

Aplikasi Streamlit ini mengubah ide mentah menjadi naskah video panjang lengkap dengan *prompt* visual dan narasi, layaknya alat produksi konten otomatis.

## Fitur
- Custom Niche, Durasi, dan Tone Suara.
- Output JSON terstruktur yang diterjemahkan langsung ke antarmuka kartu (*scene cards*).
- Menghasilkan Prompt Visual (Bahasa Inggris) dan Script Voice Over (Bahasa Indonesia).

## Cara Menjalankan Secara Lokal
1. Clone repositori ini.
2. Install dependensi: `pip install -r requirements.txt`
3. Jalankan aplikasi: `streamlit run app.py`
